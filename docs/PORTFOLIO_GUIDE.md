# 포트폴리오 기능 사용 가이드

## 🎯 테스트 완료 항목

### ✅ 1. 개인화 RAG 챗봇
**위치**: Streamlit 앱 → "💼 내 포트폴리오" 탭 → "💬 개인화 챗봇" 서브탭

**테스트 방법**:
1. Streamlit 앱 실행 (`streamlit run app.py`)
2. "💼 내 포트폴리오" 탭 클릭
3. "💬 개인화 챗봇" 서브탭 클릭
4. 다음 질문들을 입력해보세요:

```
내가 보유한 AI 종목들 어때?
엔비디아 전망은?
빅테크 기업들 실적은?
반도체 시장 전망은?
```

**예상 결과**:
- 답변 하단에 **"💼 보유 중인 관련 종목"** 섹션이 표시됨
- 관련 종목의 보유 수량, 수익률이 함께 표시됨
- YouTube 영상 소스도 함께 제공됨

---

### ✅ 3. 포트폴리오 시각화 & 섹터 자동 분류
**위치**: Streamlit 앱 → "💼 내 포트폴리오" 탭 → "📊 시각화" 서브탭

**기능**:
1. **섹터 자동 분류**: yfinance API를 통해 각 종목의 섹터 및 산업군을 자동으로 판별합니다.
2. **다양한 인터랙티브 차트**:
   - **섹터 비중 (Pie)**: 포트폴리오의 섹터별 분산도를 확인합니다.
   - **종목별 수익률 (Bar)**: 어떤 종목이 가장 높은 성과를 내고 있는지 한눈에 비교합니다.
   - **포트폴리오 트리맵 (Treemap)**: 종목 비중(크기)과 수익률(색상)을 동시에 분석합니다.
   - **할당 구조 (Sunburst)**: 섹터에서 종목으로 이어지는 계층적 자산 배분을 시각화합니다.
   - **섹터별 성과 (Performance)**: 섹터 비중 대비 평균 수익률을 산점도로 표시하여 효율성을 평가합니다.

**테스트 방법**:
1. "💼 내 포트폴리오" 탭 클릭
2. "📊 시각화" 서브탭 클릭
3. 데이터 로드 및 섹터 분류가 완료될 때까지 잠시 기다립니다 (최초 실행 시 약 5-10초 소요).

---

### ✅ 4. 알림 시스템
**위치**: Streamlit 앱 → "💼 내 포트폴리오" 탭 → "🔔 알림" 서브탭

**테스트 방법**:
1. "💼 내 포트폴리오" 탭 클릭
2. "🔔 알림" 서브탭 클릭
3. "🔍 알림 확인" 버튼 클릭
4. 결과 확인 (약 1-2분 소요)

**예상 결과**:
- 가격 변동 알림 (±10% 이상 종목)
- 뉴스 알림 (최근 7일 내 중요 뉴스)
- 손절 알림 (수익률 -15% 이하)
- 익절 알림 (수익률 +30% 이상)
- 경고 (수익률 -10% ~ -15%)

## 🚀 추가 기능 테스트

### 3. 포트폴리오 분석
**위치**: "💼 내 포트폴리오" 탭 → "🚀 분석 실행" 버튼

**테스트 방법**:
1. "💼 내 포트폴리오" 탭에서 "🚀 분석 실행" 클릭
2. 각 종목별 AI 피드백 확인
3. RAG 검색 결과 확인

### 4. 리밸런싱 제안
**위치**: "💼 내 포트폴리오" 탭 → "⚖️ 리밸런싱" 서브탭

**테스트 방법**:
1. "⚖️ 리밸런싱" 서브탭 클릭
2. "📊 리밸런싱 분석" 버튼 클릭
3. 섹터별 비중 및 제안 확인

### 5. 실시간 가격 업데이트
**위치**: "💼 내 포트폴리오" 탭 → "📡 가격 업데이트" 버튼

**테스트 방법**:
1. "📡 가격 업데이트" 버튼 클릭
2. yfinance로 실시간 가격 자동 업데이트
3. 업데이트된 가격 확인
