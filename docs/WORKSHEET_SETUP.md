# 구글 시트 워크시트 설정 가이드

## 중요: 실행 전 필수 작업

프로그램을 실행하기 전에 구글 스프레드시트에 **2개의 워크시트(탭)**를 미리 만들어야 합니다.

### 1단계: 구글 시트 열기
기존에 사용하던 구글 시트를 엽니다:
`https://docs.google.com/spreadsheets/d/14o_iIyXCCNJrwYbxD-TQgfhu139fuGiWlvFo46PbQ7k/edit`

### 2단계: 워크시트 생성

#### 첫 번째 탭: `Youtube_Log`
1. 시트 하단의 `+` 버튼 클릭 (또는 기존 시트 이름 우클릭 → 이름 바꾸기)
2. 이름을 정확히 `Youtube_Log`로 입력

#### 두 번째 탭: `Market_Log`
1. 다시 `+` 버튼 클릭
2. 이름을 정확히 `Market_Log`로 입력

### 3단계: 확인
- 시트 하단에 `Youtube_Log`와 `Market_Log` 탭이 보여야 합니다
- 철자가 정확히 일치해야 합니다 (대소문자 구분)

### 4단계: 프로그램 실행
```bash
.venv\Scripts\python.exe main.py
```

## 자동 생성되는 내용

프로그램이 처음 실행되면 각 워크시트에 자동으로 헤더가 생성됩니다:

**Youtube_Log 헤더:**
```
수집일시 | 채널명 | 영상제목 | AI_3줄요약 | 영상링크 | 업로드일자
```

**Market_Log 헤더:**
```
수집일시 | CNN지수 | CNN상태 | 코스피 | 나스닥 | 원달러환율 | 삼성전자 | 테슬라 | 엔비디아 | 비트코인
```

## 주의사항
- 워크시트 이름이 정확하지 않으면 프로그램이 자동으로 생성하지만, 기존 데이터와 분리될 수 있습니다
- 기존 데이터가 있는 시트를 사용하는 경우, 탭 이름만 변경하면 됩니다
